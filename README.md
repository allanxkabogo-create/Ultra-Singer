# UltraSinger
A Hyper-Realistic Singing Voice Synthesis Model using Diffusion Transformers and Neural Codecs.

## Goal
To achieve MOS > 4.5 and create the "Ultra-DALI" 100% Ground Truth dataset.

## Architecture
- **Tokenizer:** DAC (44.1kHz)
- **Backbone:** DiT (Diffusion Transformer)
- **Alignment:** Explicit Duration Control (Length Regulator)
